# SmartTask AI - Android

A modern Android productivity application demonstrating
clean architecture and AI-assisted task suggestions.

Tech stack:

- Kotlin
- MVVM
- Clean Architecture
- Room Database
- Retrofit API
- Coroutines / Flow
- Jetpack Compose
- Hilt Dependency Injection
- GitHub Actions CI/CD

Features:

• Task management
• AI-powered task suggestions
• Offline-first architecture
• Modern Android architecture

Purpose:

Portfolio project showcasing scalable Android development practices.
